# Usage 

```
TriggerEvent('redux_textui:ShowUI', "TEXT", "THE PROVIDED KEYBIND") 
```
or
```
exports.redux_textui:Show("TEXT", "THE PROVIDED KEYBIND")
```

to hide

```
exports.redux_textui:Hide()
```
